JAVA入门知识梳理 01

maven 是什么？

一个java中第三方jar包的管理工具。其它语言中也有类似工具，例如当前我主力使用C#中的Nuget(微软官方出品)，以及nodejs中的npm、yarn等等。maven国内使用时，需要自己搭建私服。概念还是类似于nodejs的npm注册淘宝的镜像地址，否则下载包会很慢。java类似的包管理工具还有 Gradle 等。maven还可以处理jar包的管理，还可以安装插件，支持项目打包、测试、部署。

ps: 我使用IntelliJ IDEA 工具创建了我的第一个maven demo 工程，明白了如何配置本地路径下maven,并修改setting.xml配置文件，并将maven下载地址指向内网或其它国内镜像地址。

spring 是什么？

JAVA开发中的应用程序框架，简化JAVA EE (Java Platform Enterprise Edition,经过了解类似于C#中的.NET Framework)开发，轻量，开发的时候只需要关注业务逻辑实现。主要支持功能是：

  • IoC容器 （Inversion of Control 一种编程思想：控制反转，负责控制对象的生命周期和对象间的关系）
  • DI （ Dependency Injection 依赖注入，动态的提供某个它所需要的对象）
  • AOP支持 （Aspect-Oriented Programming，面向方面编程）
  • 事务支持
  • MVC开发 （model-view-controller 编程模式）

目前我开发中使用的C# 语言编程模式是 MVVM (model-view-viewModel),我感觉这种更合理一点，controller 不需要持有model,替换成viewModel 只需要对数据进行解析，spring通过搜索结果看也是支持MVVM编程模式，进一步确定以后再说。

不过以上几点都是对编程的抽象概念，我认为不需要深入研究，明白抽象概念会用就行了。

spring boot 是什么？

spring boot 主要是让开发者能够快速使用spring应用程序，简化spring使用方法，直接使用spring时的大量配置工作去掉了。还提供了大量的第三方插件，比如Tomcat (Apache 软件基金会旗下的开源web应用服务器)等。

spring cloud 是什么？

也是基于spring开发的微服务管理框架，相当于一个管家，把所有的微服务管理了起来，里面提供了各种组件，比如springCloudBus 管理微服务之间通讯的，springCloudCondfig 管理配置的，把所有配置集中管理，别东一个西一个，马上就乱了。

mybatis 是什么？

一个数据持久化框架，半自动化ORM （Object Relational Mapping 关系数据库映射模型框架） 框架，灵活、易学，通过简单xml或配置，就能将类型、接口映射为数据库中的记录。 基于maven项目，只需要在pom.xml (Project Object Model 对象模型，maven管理工具的文件)中加入以下配置：

<dependency>
   <groupId>org.mybatis</groupId>
   <artifactId>mybatis</artifactId>
   <version>x.x.x</version>
 </dependency>

maven会自动进行安装的。

MySQL 是什么？

免费开源的关系型数据库，使用量比较大，之前玩过mysql、microsoft sql server，也略懂一点SQL语句，接触的项目对数据库要求不高，除了免费，看不出来有什么特别的地方，不做过多的研究了。非关系型数据库mongoDB也搞过，目前使用感觉还是认为关系型数据库用起来更舒服一点，可能是因为接触的项目数据量都不大。

redis 是什么？

免费开源、非关系型，运行在内存的数据库。我看一般场景下都是搭配其它数据库使用，查询了解了一下，虽然它能对数据进行持久化，但是运行在内存中，不能对海量数据进行高频次读写，适用的场景有限，所以搭配其它数据库使用，当作其它数据的扩展。

其它

JAVA入门教程主要看的是廖雪峰大佬博客，我自己有C#基础，看JAVA语法和里面的抽象概念没什么难度。比如什么集合、泛型、IO、反射等等，说法都一致。毕竟C#和JAVA就像同父异母的亲兄弟，再跟着博客中的练习教程练习一下，算是入门了吧。大佬写的教程不错，博客里也有其它教程，有兴趣的可以翻一翻。

总结一下，这一周我粗浅的了解了一下JAVA工程开发中常用的几种工具，学会了基本用法，使用IntelliJ IDEA 工具（开行即用的工具，功能全面，但是和宇宙第一IDE => Visual Studio 相比还是差了亿点）写了一个spring boot 应用Demo 《Hello JAVA!》，借助这个工具创建spring boot 项目很简单，基本跟着教程文档走就行了。
